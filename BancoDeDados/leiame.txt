GUIA DE INSTALAÇÃO PARA ANALISTAS E DESENVOLVEDORES


BANCO DE DADOS
- Instalar o SQL SERVER com Management Studio 2008.
- Criar o banco com o nome de "Banco".
- Executar a query SQL que contém a estrutura das tabelas.
- A instância do banco de dados tem que ter o nome de SQLEXPRESS, caso contrário será necessário alterar a string de conexão. Para alterá-la é só mudar a função “RetornaStrConexao()” localizada na classe BLL “ConfiguracaoBLL.vb” no diretório “Business Logic Layer”.

- Inserir os registros na tabela SANGUES de forma que fique exatamente assim (o ID é incrementado automaticamente):

ID    SANGUE
1     A+
2     A-
3     B+
4     B-
5     O+
6     O-
7     AB+
8     AB-


- Inserir os registros na tabela TIPO_PESSOAS de forma que fique exatamente assim (o ID é incrementado automaticamente):

ID   TIPO
1    Doador
2    Receptor
3    Usuario


- Inserir os registros na tabela TIPO_USUARIOS de forma que fique exatamente assim (o ID é incrementado automaticamente):

ID  TIPO
1  UsuarioNormal
2  Admin


- Inserir os registros na tabela PESSOAS de forma que fique exatamente assim (o ID é incrementado automaticamente):

ID  NOME   	  TIPO_PESSOA    			 NOME_USUARIO                    SENHA                  TIPO_USUARIO
1   MASSA           1                         NULL                        NULL                        NULL
2    NULL             3                       admin                 rnDc+ERMD9Q=                       2

(os campos não citados podem ser deixados como NULL)
(o usuário e senha acima são padrão, depois tem que alterar. Para usuário e senha use "admin").


- Inserir os registros na tabela CONFIGURACOES de forma que fique exatamente assim (o ID é incrementado automaticamente): 

NOME_CONF   VALOR_CONF
CaminhoRel      {Caminho da aplicação + "\". Exemplo: C:\Software\}
NomeEstab                                                                                                  
EnderEstab    
CEPEstab                                                                                                              
CidadeEstab                                                                                                          
EstadoEstab                                                                                                           
ValidarCPFCNPJ        0                                                                                         
PeriodoMinimoDoacoes      90                                                                               

APLICAÇÃO
- Instalar o servidor ASP.Net ou o Visual Studio 2010.
- Instalar o Crystal Reports for Visual Studio
- Copiar a pasta do programa (pasta "Software") para o local desejado.
